# To start app

1. Start Docker
2. Start GQL server(localhost::4000)
`npm run start`

# Stack

- GQL server: graphql-yoga
- Live reload: nodemon
- ID generator: uuid
- Object spreader: babel-plugin-transform-object-rest-spread
- GraphQL binding for Prisma: prisma-binding
- GraphQL cli: graphql-cli
